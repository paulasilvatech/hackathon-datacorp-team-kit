# Cheat Sheets

> Tarjetas de referencia rápida de una página, diseñadas para imprimir y dejar en la mesa de cada equipo.

## Cómo pensar en esto

Las cheat sheets no son tutoriales — son **referencia bajo presión**. Cuando estés atascado a las 16:00 y necesites decidir si esto va en Chat, Edits o Agent, no tienes 10 minutos para leer documentación. Tienes 10 segundos para mirar la tarjeta correcta. Por eso son densas, tabulares y sin floritura.
